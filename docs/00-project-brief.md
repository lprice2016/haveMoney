# 等我有钱了项目简报

> 当前有效方向见 `docs/11-product-decision-ledger.md`。本简报已同步为最新定位：私人管家式有钱爽剧。早期“虚拟富豪人生模拟器”“无限黑卡”“虚拟额度”等表述不再作为产品主线。

## 一句话

等我有钱了是一款私人管家式有钱爽剧生成器：用户不再等待“等我有钱了”，而是输入一个愿望，进入“我已经有钱了”的状态，被一个专业管家团队认真安排、服务、记住和捧到位。

## 核心用户洞察

很多普通人都有一个“有钱梦”：周游世界、买豪车豪宅、住高端酒店、吃高端餐厅、给家人更好的生活、逃离当前压力。但现实里，绝大多数人很难真的拥有想象中的财富。这个落差本身就是情绪入口。

等我有钱了不解决真实财富问题，它解决一个更轻、更直接的问题：让用户暂时进入一种“我已经有钱了，并且世界开始认真对待我”的心理状态。

## 产品承诺

- 把用户的一句“等我有钱了...”变成一段短、准、有递进、有高潮的有钱爽剧。
- 用管家式语气承接愿望，避免冷冰冰拒绝。
- 用具体服务细节制造爽感，而不是堆金额和品牌。
- 让用户感到被识别、被优先、被预判、被见证。
- 生成适合截图、发群、接梗的今日富豪战报。

## 非目标

- 不提供真实理财建议。
- 不提供贷款、借钱、投资、博彩、返利或真实赚钱承诺。
- 不制造真实阶层歧视和羞辱。
- 不诱导用户为了模仿虚拟体验进行现实高消费。

## 核心循环

1. 进入：选择“我有一个愿望”或“管家来安排”。
2. 定调：输入昵称、称呼、虚拟形象或直接跳过。
3. 开场：黑屏后进入“生活设定已重写”的管家开场。
4. 爽剧：系统生成入场、优先、见证、改规则、高潮。
5. 战报：生成今日富豪战报，适合截图或分享。
6. 回访：下一次打开时继续昨天安排，或让管家换一种爽法。

## MVP 功能

- 愿望输入：用户写下“等我有钱了，我要...”。
- 管家来安排：用户不输入具体愿望，由系统主动安排今日有钱人生。
- 风险转译：色情、赌博、违法、真实金融误导等输入转成安全但不扫兴的表达。
- 短剧分镜：生成有递进的 60 秒爽剧。
- 今日富豪战报：生成可截图、可转发、可接梗的结果。
- 手动 Q5 测试：先用真人反馈验证内容，再决定是否做 H5 和服务端。

## 第一阶段成功标准

- 用户能在 30 秒内明白玩法。
- 用户第一次生成后产生明显爽感或会心一笑。
- 用户愿意写下至少 1 个自己的有钱梦。
- 用户愿意把一次爽剧结果截图分享给朋友。
- 产品边界清晰，不被理解成真实金融产品。
