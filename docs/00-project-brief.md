# 等我有钱了项目简报

## 一句话

等我有钱了是一款“虚拟富豪人生模拟器”：用户不再等待“等我有钱了”，而是直接进入“我已经有钱了”的状态，体验花钱、选择、炫耀、烦恼和幻想成真的快感。

## 核心用户洞察

很多普通人都有一个“有钱梦”：周游世界、买豪车豪宅、住高端酒店、吃高端餐厅、给家人更好的生活、逃离当前压力。但现实里，绝大多数人很难真的拥有想象中的财富。这个落差本身就是情绪入口。

等我有钱了不解决真实财富问题，它解决一个更轻、更直接的问题：让用户暂时进入一种“我已经有钱了”的心理状态。

## 产品承诺

- 快速获得虚拟财富身份。
- 低门槛体验高消费场景。
- 用夸张数字和细节反馈制造爽感。
- 用“钱太多也麻烦”的轻喜剧反差增加记忆点。
- 让用户把自己的有钱梦保存、分享、继续扩写。

## 非目标

- 不提供真实理财建议。
- 不提供贷款、借钱、投资、博彩、返利或真实赚钱承诺。
- 不制造真实阶层歧视和羞辱。
- 不诱导用户为了模仿虚拟体验进行现实高消费。

## 核心循环

1. 进入：获得一个虚拟有钱身份、身价和账户额度。
2. 选择：从愿望、场景、突发事件中选择“花钱目标”。
3. 消费：用虚拟黑卡完成夸张消费。
4. 反馈：获得账单、旁白、状态提升、社交截图或富人烦恼。
5. 沉淀：收藏资产、生成故事、更新愿望清单。
6. 回访：每日财富事件、限时场景、朋友挑战或新烦恼。

## MVP 功能

- 身价生成器：随机生成“今日资产”“现金流”“身份标签”。
- 无限黑卡：展示额度、刷卡动效和消费记录。
- 愿望清单：用户写下“等我有钱了我要...”。
- 奢华场景库：豪车、豪宅、环球旅行、高端酒店、高端餐厅、私人服务。
- 消费反馈：每次消费给出戏剧化结果和情绪奖励。
- 富人烦恼：如“私人飞机停哪”“酒庄又送来十箱酒”“管家请假了”。
- 分享卡片：生成“今天我花了多少钱”的娱乐截图。

## 第一阶段成功标准

- 用户能在 30 秒内明白玩法。
- 用户第一次消费后产生明显爽感或会心一笑。
- 用户愿意写下至少 1 个自己的有钱梦。
- 用户愿意把一次消费结果截图分享给朋友。
- 产品边界清晰，不被理解成真实金融产品。
