# 上架与品牌决策

## 当前结论

产品正式名定为：等我有钱了。

副标题建议：虚拟富豪人生模拟器。

仓库名继续使用 `haveMoney`，作为工程代号。

## 上架策略

第一版不要按“游戏”定位，建议按“娱乐 / 生活方式 / 互动模拟”定位。

理由：

- 产品核心是情绪价值、愿望表达、虚拟消费和可分享故事，不是竞技、关卡、输赢或概率奖励。
- App Store 可以上架这种娱乐模拟类 App，但要避免真实金融、博彩、误导性财富承诺和低价值玩笑应用。
- 国内安卓市场如果被识别为网络游戏，容易触发版号、实名、防沉迷、未成年人限制等高门槛。

## 审核红线

必须避免：

- 真实赚钱、提现、返利、投资收益、贷款、额度申请。
- “黑卡真实开通”“财富自由方案”“躺赚”等可能被理解为金融承诺的表达。
- 抽卡、老虎机、下注、彩票、概率爆奖等博彩化机制。
- 让用户充值购买“虚拟现金包”作为核心玩法。
- 诱导现实高消费、借贷或攀比。

必须明确：

- 所有资产、身份、金额和消费均为虚拟娱乐模拟。
- 不提供理财、借贷、投资或真实资产管理服务。
- 分享卡应偏幽默和剧情，不展示可被误解为真实银行、真实账单或真实支付成功的内容。

## 命名理由

“等我有钱了”本身就是用户熟悉的口头禅，传播时天然带句式：

- 等我有钱了，我要买下这家酒店。
- 等我有钱了，我要坐私人飞机去吃早餐。
- 等我有钱了，我先给自己放十年假。

相比“有钱花”，新名字更像娱乐表达，少了借贷和金融产品联想，更适合上架和传播。
