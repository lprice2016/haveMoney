# 等我有钱了

私人管家式有钱爽剧生成器。仓库名暂用 `haveMoney`。

当前阶段：GStack/Codex 产品验证阶段。

本项目暂缓 App 主线开发，先验证“等我有钱了”这个内容和传播机制是否成立。核心问题不是技术能不能做，而是用户看到生成结果后会不会觉得爽、想继续、想截图、转发、接梗。
