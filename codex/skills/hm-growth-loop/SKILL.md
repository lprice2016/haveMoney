# HM Growth Loop

## Purpose

Design growth experiments before product build-out. Prove users want to share or reply.

## Required Context

Read as needed:

- `docs/05-content-validation-plan.md` for the current test protocol.
- `docs/04-plan-ceo-review.md` for success standards.
- `docs/08-risk-review.md` for safety boundaries.

## Core Loop

The preferred early loop is:

1. User writes "等我有钱了，我要..."
2. Product generates a sharp, funny, or tender result.
3. User shares it.
4. Friend replies with their own "等我有钱了..."
5. Product captures the next prompt.

## Experiment Types

- WeChat group text test.
-朋友圈 screenshot test.
- Xiaohongshu title test.
- Side-by-side copy ranking.
- Prompt completion test: ask friends to finish the sentence.

## Metrics

Use lightweight evidence:

- Share intent: "would you send this?"
- Reply intent: "what would you write?"
- Screenshot worthiness.
- Confusion rate.
- Risky interpretation rate.
